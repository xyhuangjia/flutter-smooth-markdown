# 使用增强 UI 组件

本文档介绍如何在您的应用中使用 Flutter Smooth Markdown 的增强 UI 组件。

## 概述

增强组件提供了更好的视觉效果和用户体验：

| 组件 | 标准版 | 增强版 |
|------|--------|--------|
| 代码块 | ❌ 基础样式 | ✅ 复制按钮、语言标签、阴影效果 |
| 引用块 | ❌ 简单边框 | ✅ 引号图标、渐变背景、阴影 |
| 链接 | ❌ 下划线 | ✅ 悬停动画、外部链接图标 |
| 标题 | ❌ 纯文本 | ✅ 彩色标记、渐变边框 |

## 快速开始

### 方式一：使用自定义构建器注册表

```dart
import 'package:flutter_smooth_markdown/flutter_smooth_markdown.dart';
import 'package:flutter_smooth_markdown/src/renderer/builders/enhanced_code_block_builder.dart';
import 'package:flutter_smooth_markdown/src/renderer/builders/enhanced_blockquote_builder.dart';
import 'package:flutter_smooth_markdown/src/renderer/builders/enhanced_link_builder.dart';
import 'package:flutter_smooth_markdown/src/renderer/builders/enhanced_header_builder.dart';

// 创建自定义渲染器
final customRenderer = MarkdownRenderer(
  styleSheet: MarkdownStyleSheet.light(),
);

// 注册增强组件
customRenderer.builderRegistry
  ..register('code_block', const EnhancedCodeBlockBuilder())
  ..register('blockquote', const EnhancedBlockquoteBuilder())
  ..register('link', const EnhancedLinkBuilder())
  ..register('header', const EnhancedHeaderBuilder());

// 使用自定义渲染器
final nodes = MarkdownParser().parse(markdownText);
final widget = customRenderer.render(nodes);
```

### 方式二：使用自定义 Widget 构建器

```dart
// 代码块使用增强版本
SmoothMarkdown(
  data: markdownText,
  codeBuilder: (code, language) {
    return EnhancedCodeBlockWidget(
      code: code,
      language: language,
      showCopyButton: true,
      showLanguageTag: true,
    );
  },
)
```

## 组件配置

### 1. 增强代码块

```dart
const EnhancedCodeBlockBuilder(
  showCopyButton: true,    // 显示复制按钮
  showLanguageTag: true,   // 显示语言标签
)
```

**功能：**
- 悬停时显示复制按钮
- 点击复制代码到剪贴板
- 显示 "Copied!" 确认消息
- 语言标签（如 DART、JAVASCRIPT）
- 悬停时阴影效果
- 可选择的文本

**样式定制：**
```dart
// 在 MarkdownStyleSheet 中设置
codeBlockDecoration: BoxDecoration(
  color: Colors.grey[100],
  borderRadius: BorderRadius.circular(8),
),
codeBlockStyle: TextStyle(
  fontFamily: 'monospace',
  fontSize: 14,
),
```

### 2. 增强引用块

```dart
const EnhancedBlockquoteBuilder(
  showIcon: true,  // 显示引号图标
)
```

**功能：**
- 引号图标装饰
- 渐变背景
- 彩色左侧边框
- 微妙阴影
- 圆角右侧

**样式示例：**
```dart
// 亮色主题
> 这是一个引用块
> 支持多行文本

// 暗色主题
> 自动适配暗色主题
> 保持优雅的视觉效果
```

### 3. 增强链接

```dart
const EnhancedLinkBuilder(
  showExternalIcon: true,  // 显示外部链接图标
)
```

**功能：**
- 悬停时下划线动画
- 下划线宽度和透明度过渡
- 外部链接自动添加图标
- 平滑动画效果（200ms）
- 鼠标指针变化

**示例：**
```markdown
[Flutter](https://flutter.dev)  // 显示外部链接图标
[内部链接](#section)            // 不显示图标
```

### 4. 增强标题

```dart
const EnhancedHeaderBuilder(
  showBottomBorder: true,  // H1/H2 显示底部边框
)
```

**功能：**
- H1/H2 左侧彩色竖线
- 渐变色装饰
- 底部渐变边框
- 优化的间距

**层级效果：**
```markdown
# H1 标题 - 竖线 + 底部边框
## H2 标题 - 竖线 + 底部边框
### H3 标题 - 仅竖线
#### H4-H6 标题 - 纯文本
```

## 完整示例

### 基础使用

```dart
import 'package:flutter/material.dart';
import 'package:flutter_smooth_markdown/flutter_smooth_markdown.dart';

class EnhancedMarkdownPage extends StatelessWidget {
  final String markdown = '''
# 欢迎使用增强 UI

试试这些功能：

```dart
// 悬停查看复制按钮
void main() {
  print('Hello Enhanced UI!');
}
\```

> 这是一个美丽的引用块
> 注意引号图标和渐变背景

查看 [Flutter 文档](https://flutter.dev) 了解更多！
''';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Enhanced UI Demo')),
      body: SingleChildScrollView(
        padding: EdgeInsets.all(16),
        child: SmoothMarkdown(
          data: markdown,
          styleSheet: MarkdownStyleSheet.light(),
        ),
      ),
    );
  }
}
```

### 自定义主题

```dart
final customStyleSheet = MarkdownStyleSheet.light().copyWith(
  // 代码块样式
  codeBlockDecoration: BoxDecoration(
    gradient: LinearGradient(
      colors: [Colors.blue[50]!, Colors.purple[50]!],
    ),
    borderRadius: BorderRadius.circular(12),
    boxShadow: [
      BoxShadow(
        color: Colors.blue.withValues(alpha: 0.1),
        blurRadius: 8,
        offset: Offset(0, 2),
      ),
    ],
  ),
  codeBlockStyle: TextStyle(
    fontFamily: 'Fira Code',
    fontSize: 14,
    height: 1.5,
  ),

  // 链接样式
  linkStyle: TextStyle(
    color: Colors.purple[700],
    fontWeight: FontWeight.w500,
  ),

  // 标题样式
  h1Style: TextStyle(
    fontSize: 36,
    fontWeight: FontWeight.bold,
    letterSpacing: -0.5,
  ),
);

SmoothMarkdown(
  data: markdown,
  styleSheet: customStyleSheet,
)
```

## 性能优化

### 1. 避免重复创建

```dart
class MyWidget extends StatelessWidget {
  // ✅ 好：复用 builder
  static const _codeBuilder = EnhancedCodeBlockBuilder();

  @override
  Widget build(BuildContext context) {
    // 使用 _codeBuilder
  }
}
```

### 2. 按需启用功能

```dart
// 如果不需要复制功能，可以禁用
const EnhancedCodeBlockBuilder(
  showCopyButton: false,  // 节省资源
  showLanguageTag: true,
)
```

### 3. 大量内容时使用虚拟滚动

```dart
// 对于很长的文档
ListView.builder(
  itemCount: sections.length,
  itemBuilder: (context, index) {
    return SmoothMarkdown(
      data: sections[index],
      // ...
    );
  },
)
```

## 常见问题

### Q: 复制按钮在移动端不显示？

A: 移动端使用触摸操作，没有"悬停"概念。建议在移动端：
```dart
const EnhancedCodeBlockBuilder(
  showCopyButton: true,  // 始终显示
)
```

### Q: 如何禁用外部链接图标？

A: 设置 `showExternalIcon: false`：
```dart
const EnhancedLinkBuilder(
  showExternalIcon: false,
)
```

### Q: 可以混用标准和增强组件吗？

A: 可以！根据需要选择性使用：
```dart
// 只增强代码块
registry.register('code_block', const EnhancedCodeBlockBuilder());
// 其他使用标准组件
registry.register('blockquote', const BlockquoteBuilder());
```

### Q: 如何自定义复制按钮样式？

A: 目前需要继承 `EnhancedCodeBlockBuilder` 并覆盖 `_EnhancedCodeBlockWidget`。我们计划在未来版本添加更多自定义选项。

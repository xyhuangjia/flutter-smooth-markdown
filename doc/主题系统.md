# Flutter Smooth Markdown 主题系统

本文档介绍 Flutter Smooth Markdown 的主题系统及其使用方法。

## 概述

Flutter Smooth Markdown 提供了强大且灵活的主题系统，支持：

- ✅ 内置 Light/Dark 模式
- ✅ 预设主题（GitHub、VS Code 风格）
- ✅ 完全自定义主题
- ✅ 自动适配 Flutter 主题
- ✅ 细粒度样式控制

## 快速开始

### 1. 使用默认主题

```dart
// 亮色主题
SmoothMarkdown(
  data: markdownText,
  styleSheet: MarkdownStyleSheet.light(),
)

// 暗色主题
SmoothMarkdown(
  data: markdownText,
  styleSheet: MarkdownStyleSheet.dark(),
)
```

### 2. 自动适配 Flutter 主题

```dart
SmoothMarkdown(
  data: markdownText,
  styleSheet: MarkdownStyleSheet.fromTheme(Theme.of(context)),
)
```

这会根据当前 Flutter 主题的 brightness 自动选择亮色或暗色主题。

### 3. 使用预设主题

#### GitHub 风格

```dart
// GitHub 亮色
SmoothMarkdown(
  data: markdownText,
  styleSheet: MarkdownStyleSheet.github(),
)

// GitHub 暗色
SmoothMarkdown(
  data: markdownText,
  styleSheet: MarkdownStyleSheet.github(brightness: Brightness.dark),
)
```

#### VS Code 风格

```dart
// VS Code 亮色
SmoothMarkdown(
  data: markdownText,
  styleSheet: MarkdownStyleSheet.vscode(),
)

// VS Code 暗色
SmoothMarkdown(
  data: markdownText,
  styleSheet: MarkdownStyleSheet.vscode(brightness: Brightness.dark),
)
```

## 自定义主题

### 完全自定义

```dart
final customTheme = MarkdownStyleSheet(
  // 文本样式
  textStyle: TextStyle(fontSize: 16, color: Colors.black),

  // 标题样式
  h1Style: TextStyle(fontSize: 32, fontWeight: FontWeight.bold),
  h2Style: TextStyle(fontSize: 28, fontWeight: FontWeight.bold),
  h3Style: TextStyle(fontSize: 24, fontWeight: FontWeight.bold),

  // 代码块样式
  codeBlockStyle: TextStyle(
    fontFamily: 'monospace',
    fontSize: 14,
  ),
  codeBlockDecoration: BoxDecoration(
    color: Colors.grey[100],
    borderRadius: BorderRadius.circular(8),
    border: Border.all(color: Colors.grey[300]!),
  ),

  // 链接样式
  linkStyle: TextStyle(
    color: Colors.blue,
    decoration: TextDecoration.underline,
  ),

  // 引用块样式
  blockquoteDecoration: BoxDecoration(
    border: Border(
      left: BorderSide(color: Colors.grey, width: 4),
    ),
    color: Colors.grey[50],
  ),

  // 间距设置
  blockSpacing: 16,
  listIndent: 24,
);

SmoothMarkdown(
  data: markdownText,
  styleSheet: customTheme,
)
```

### 基于现有主题修改

```dart
final modifiedTheme = MarkdownStyleSheet.light().copyWith(
  h1Style: TextStyle(
    fontSize: 40,
    fontWeight: FontWeight.bold,
    color: Colors.purple,
  ),
  linkStyle: TextStyle(
    color: Colors.green,
    decoration: TextDecoration.none,
  ),
);

SmoothMarkdown(
  data: markdownText,
  styleSheet: modifiedTheme,

```

## 主题属性详解

### 文本样式

| 属性 | 说明 | 类型 |
|------|------|------|
| `textStyle` | 默认文本样式 | `TextStyle?` |
| `h1Style` - `h6Style` | 标题样式 (H1-H6) | `TextStyle?` |
| `paragraphStyle` | 段落样式 | `TextStyle?` |
| `boldStyle` | 粗体样式 | `TextStyle?` |
| `italicStyle` | 斜体样式 | `TextStyle?` |
| `strikethroughStyle` | 删除线样式 | `TextStyle?` |
| `linkStyle` | 链接样式 | `TextStyle?` |
| `codeBlockStyle` | 代码块文本样式 | `TextStyle?` |
| `inlineCodeStyle` | 行内代码样式 | `TextStyle?` |
| `blockquoteStyle` | 引用文本样式 | `TextStyle?` |
| `listBulletStyle` | 列表标记样式 | `TextStyle?` |

### 装饰样式

| 属性 | 说明 | 类型 |
|------|------|------|
| `codeBlockDecoration` | 代码块装饰 | `BoxDecoration?` |
| `blockquoteDecoration` | 引用块装饰 | `BoxDecoration?` |
| `tableBorder` | 表格边框 | `TableBorder?` |
| `horizontalRuleColor` | 水平分隔线颜色 | `Color?` |
| `horizontalRuleThickness` | 水平分隔线粗细 | `double?` |

### 间距设置

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| `blockSpacing` | 块元素间距 | `double?` | 16 |
| `listIndent` | 列表缩进 | `double?` | 24 |
| `blockquotePadding` | 引用块内边距 | `EdgeInsets?` | 16,12 |
| `codeBlockPadding` | 代码块内边距 | `EdgeInsets?` | 12 |
| `tableCellPadding` | 表格单元格内边距 | `EdgeInsets?` | 8 |

## 响应式主题

### 根据系统主题自动切换

```dart
class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData.light(),
      darkTheme: ThemeData.dark(),
      home: Builder(
        builder: (context) {
          return SmoothMarkdown(
            data: markdownText,
            styleSheet: MarkdownStyleSheet.fromTheme(
              Theme.of(context),
            ),
          );
        },
      ),
    );
  }
}
```

### 手动控制主题

```dart
class MyWidget extends StatefulWidget {
  @override
  State<MyWidget> createState() => _MyWidgetState();
}

class _MyWidgetState extends State<MyWidget> {
  Brightness _brightness = Brightness.light;

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        // 主题切换按钮
        IconButton(
          icon: Icon(
            _brightness == Brightness.dark
                ? Icons.light_mode
                : Icons.dark_mode,
          ),
          onPressed: () {
            setState(() {
              _brightness = _brightness == Brightness.dark
                  ? Brightness.light
                  : Brightness.dark;
            });
          },
        ),

        // Markdown 内容
        SmoothMarkdown(
          data: markdownText,
          styleSheet: MarkdownStyleSheet.fromBrightness(_brightness),
        ),
      ],
    );
  }
}
```

## 主题最佳实践

### 1. 使用主题常量

```dart
class AppThemes {
  static final light = MarkdownStyleSheet.light();
  static final dark = MarkdownStyleSheet.dark();
  static final github = MarkdownStyleSheet.github();
  static final githubDark = MarkdownStyleSheet.github(
    brightness: Brightness.dark,
  );
}

// 使用
SmoothMarkdown(
  data: markdownText,
  styleSheet: AppThemes.github,
)
```

### 2. 集成到应用主题

```dart
class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(
        brightness: Brightness.light,
        primarySwatch: Colors.blue,
        // 其他主题配置...
      ),
      builder: (context, child) {
        return MarkdownTheme(
          data: MarkdownStyleSheet.fromTheme(Theme.of(context)),
          child: child!,
        );
      },
    );
  }
}
```

### 3. 性能优化

```dart
class MyWidget extends StatelessWidget {
  // 将主题缓存为静态常量，避免重复创建
  static final _styleSheet = MarkdownStyleSheet.light();

  @override
  Widget build(BuildContext context) {
    return SmoothMarkdown(
      data: markdownText,
      styleSheet: _styleSheet,
    );
  }
}
```

## 常见问题

### Q: 如何让主题跟随系统深色模式？

A: 使用 `MarkdownStyleSheet.fromTheme(Theme.of(context))`，配合 MaterialApp 的 theme 和 darkTheme。

### Q: 可以为不同的 Markdown 元素使用完全不同的字体吗？

A: 可以！每个样式属性都支持独立的 `fontFamily` 设置。

### Q: 如何实现平滑的主题切换动画？

A: 使用 `AnimatedSwitcher` 包裹 `SmoothMarkdown` widget。

```dart
AnimatedSwitcher(
  duration: Duration(milliseconds: 300),
  child: SmoothMarkdown(
    key: ValueKey(_currentTheme),
    data: markdownText,
    styleSheet: _currentTheme,
  ),
)
```
